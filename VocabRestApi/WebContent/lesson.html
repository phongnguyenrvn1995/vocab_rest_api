Lesson API

Get lesson list by status ID
Method: GET
Path: /VocabRestApi/vocab_api/lesson/get_by_status/{id}
Produces: APPLICATION_JSON
Get lesson list by course ID
Method: GET
Path: /VocabRestApi/vocab_api/lesson/get_by_course/{id}
Produces: APPLICATION_JSON
Get all lesson in database
Method: GET
Path: /VocabRestApi/vocab_api/lesson/gets
Produces: APPLICATION_JSON
Get a lesson detail by its ID
Method: GET
Path: /VocabRestApi/vocab_api/lesson/get/{id}
Produces: APPLICATION_JSON
Delete a lesson by its ID
Method: DELETE
Path: /VocabRestApi/vocab_api/lesson/delete/{id}
Produces: APPLICATION_JSON
Insert a lesson into database
Method: PUT
Path: /VocabRestApi/vocab_api/lesson/save
Produces: APPLICATION_JSON
Parameters:
Type:
x-www-form-urlencoded
Integer: lesson_course
String: lesson_name
Integer: lesson_status
Update a lesson in database
Method: POST
Path: /VocabRestApi/vocab_api/lesson/update
Produces: APPLICATION_JSON
Parameters:
Type:
x-www-form-urlencoded
Integer: lesson_id
Integer: lesson_course
String: lesson_name
Integer: lesson_status
