Vocab API

Method: GET
Path: /VocabRestApi/vocab_api/vocab/gets_by_type/{id}
Produces: APPLICATION_JSON
Get Vocab List by Lesson ID
Method: GET
Path: /VocabRestApi/vocab_api/vocab/gets_by_lesson/{id}
Produces: APPLICATION_JSON
Get All Vocab in database
Method: GET
Path: /VocabRestApi/vocab_api/vocab/gets
Produces: APPLICATION_JSON
Get a Vocab detail by its ID
Method: GET
Path: /VocabRestApi/vocab_api/vocab/get/{id}
Produces: APPLICATION_JSON
DELETE a Vocab detail by its ID
Method: DELETE
Path: /VocabRestApi/vocab_api/vocab/delete/{id}
Produces: APPLICATION_JSON

Method: PUT
Path: /VocabRestApi/vocab_api/vocab/save
Produces: APPLICATION_JSON
Parameters:
Type:
