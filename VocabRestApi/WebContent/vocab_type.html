Vocab Type API

Get All Vocab Type List
Method: GET
Path: /VocabRestApi/vocab_api/vocab_type/gets
Produces: APPLICATION_JSON
Get a Vocab Type by its ID
Method: GET
Path: /VocabRestApi/vocab_api/vocab_type/get/{id}
Produces: APPLICATION_JSON
Delete a Vocab Type by its ID
Method: DELETE
Path: /VocabRestApi/vocab_api/vocab_type/delete/{id}
Produces: APPLICATION_JSON
Insert a Vocab Type into database
Method: PUT
Path: /VocabRestApi/vocab_api/vocab_type/save
Produces: APPLICATION_JSON
Parameters:
Type:
x-www-form-urlencoded
String: vocab_type_name

Method: POST
Path: /VocabRestApi/vocab_api/vocab_type/update
Produces: APPLICATION_JSON
Parameters:
Type:
x-www-form-urlencoded
Integer: vocab_type_id
